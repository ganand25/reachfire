Plan and implement a new feature. Follow this process:

1. Read CLAUDE.md for project conventions
2. Create a plan (what files to change, what tests to write)
3. Present the plan and wait for approval
4. Implement the feature following all coding conventions
5. Write tests (unit + integration as appropriate)
6. Run all tests and fix any failures
7. Run lint and format checks
8. Create a descriptive commit
